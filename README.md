# Triple Espresso

Nombre del proyecto: Pagina web de una cafeteria

Descripcion del proyecto: Permite a los usuarios acceder y conocer la pagina web de la cafeteria Triple Peaks y reservar una mesa antes de su llegada. se uso HTML y CSS para su creacion con diversos archivos.

Planes de mejora: Agregar formas de contacto directos como whatsapp, numero telefonico o piner un chat directo en la pagina. Mejorar el UI agregando el menu de bebidas y alimentos.
